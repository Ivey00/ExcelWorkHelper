# Excel Reference Matcher (ExcelWorkHelper)

A web-based tool for intelligent Excel data processing, featuring reference-based fusion and product statistics generation.

## Features

### 🔄 Fusion by Reference
- Upload two Excel files (XLSX, XLS, CSV)
- Intelligent matching based on reference similarity
- Adjustable similarity threshold (50-100%)
- Merge data from price table into main table
- Download merged Excel file

### 📊 Product Statistics
- Generate statistics by brand/product
- Visual column mapping interface
- Comprehensive data analysis

## How to Use

1. **Open the HTML file** in a modern web browser
2. **Choose a tab**: Fusion or Statistics
3. **Upload files** by clicking or dragging into the drop zones
4. **Configure columns** if needed
5. **Run the process** and download results

## Requirements

- Modern web browser with JavaScript enabled
- No server required - runs entirely in the browser
- Supports Excel files (.xlsx, .xls) and CSV files

## Browser Compatibility

Works in all modern browsers that support:
- File API
- Web Workers (for processing)
- ES6+ JavaScript features

## Privacy

All processing happens locally in your browser. Files are not uploaded to any server.
